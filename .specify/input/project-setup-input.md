## プロジェクト概要

- **プロジェクト名**:LSystem
- **一言で説明すると**:補助金申請フォーマット自動作成システム
- **解決したい課題**:補助金申請支援の際に多くドキュメント、特に顧客の事業計画書のようなものを作成するが、この作成をある程度AIを駆使して自動化したい。そしてこのシステムを他の補助金申請業者に売りたい。行政書士とか

### 機能 1: [項目作成機能]（必須：最低1機能）

#### 概要
- **何をする機能？**:事業計画書内のセクション(今回の事業実施の背景)などがそれぞれテンプレとして切り分けられており、その各項目の内容を参考資料などを基に自動で埋めてくれる。
- **誰が使う？**:申請書作成者
- **なぜ必要？**:記述が面倒

#### データ
- **入力データ**:参考資料、会社データ
- **出力データ**:事業計画書内の対象セクションの文章
- **保存データ**:事業計画書(対象セクション記述積み)

#### 画面イメージ
- **メイン画面の要素**:/.specify/assets/wireframes/template_image.png
- **操作フロー**:セクション一覧から対象セクションを選択。関連するファイルを自動判別し、紐づけ済み、あとはメモを入力したり、追加の指示を入れて生成ボタンを押すことでセクションの内容が生成される。
