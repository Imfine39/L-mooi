## 1. Screen Overview

### 1.1 Purpose

このドキュメントは LSystem（補助金申請フォーマット自動作成システム）の全画面構成を定義します。

- 全画面の一覧と目的
- 画面遷移フロー
- 各画面の詳細レイアウト

**Note:** 画面 - Feature - API - Master の対応関係は `../matrix/cross-reference.json` で一元管理。

### 1.2 Design Principles

- シンプルで見やすい企業システム風デザイン
- 直感的な操作フロー（補助金 → 書類 → セクション）
- 作業状況が一目でわかる進捗表示
- PC 利用を前提とした画面設計

### SCR-010: テンプレート管理画面

**Purpose:** 補助金テンプレート（書類・セクション構造）の編集

**Entry Points:**
- SCR-002: メニュー選択（管理者のみ）

**Exit Points:**
- SCR-002: 戻る

**Primary Actions:**
- 補助金テンプレート選択
- 書類構造編集
- セクション追加/編集/削除

**Layout Structure:**
```
┌─────────────────────────────────────────────────────────────────┐
│  [Header]                                                       │
├─────────────────────────────────────────────────────────────────┤
│  テンプレート管理                          [+ 新規テンプレート]  │
├───────────────────────────┬─────────────────────────────────────┤
│  補助金テンプレート一覧   │  テンプレート詳細                    │
│                           │                                      │
│  > 新規事業創出補助金 ●  │  新規事業創出補助金                  │
│    ものづくり補助金       │                                      │
│    IT導入補助金           │  書類構造:                           │
│                           │  📁 事業計画書                       │
│                           │     ├ SWOT分析                      │
│                           │     ├ 事業背景                      │
│                           │     ├ 取り組み内容                  │
│                           │     └ [+ セクション追加]            │
│                           │  📁 収支計画書                       │
│                           │     └ 収益計画                      │
│                           │  [+ 書類追加]                        │
└───────────────────────────┴─────────────────────────────────────┘
```

**テンプレートの定義:**
- テンプレートとは「補助金ごとの書類・セクション構造の雛形」
- 例: 「新規事業創出補助金」のテンプレートには「事業計画書」という書類があり、その中に「SWOT分析」「事業背景」などのセクションが定義されている
- 新しい補助金を作成する際、テンプレートを選択すると書類・セクション構造が自動で作成される
- テンプレートは管理者のみが編集可能
